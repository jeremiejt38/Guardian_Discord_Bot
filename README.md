# Guardian_-Discord_Bot
Discord bot for gaming communities — manages game opt-in channels,  temporary voice channels, member onboarding and server moderation.

## Project structure

The bot source code is scaffolded in `/home/runner/work/Guardian_-Discord_Bot/Guardian_-Discord_Bot/guardian`.

## Quick start

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies: `npm install`
3. Run tests: `npm test`
4. Start bot: `npm start`
